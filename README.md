# Vocabulary Test UI

A Next.js (App Router) web UI for an English vocabulary estimation test.

## Features
- Sentence-based testing flow
- Clickable word explanations / translation assist
- Result page with scoring summary

## Tech Stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS / shadcn-ui

## Getting Started (Local)
1. Install dependencies:
   - npm:
     \
pm install\
2. Run dev server:
   \
pm run dev\
3. Open:
   http://localhost:3000/test
